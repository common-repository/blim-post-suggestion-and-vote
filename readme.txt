A simple plugin that suggests post and offer vote feature

== Description ==

1. Blim Post Suggestion and Vote (BPSV) is a plugin that suggests posts based on the category of the current post. 
2. The plugin comes fully setup, therefore, you are not required to perform any setup after activation. 
3. However, you can decide to manually, choose among the features offered by BPSV. You simply go to the general settings menu then click on Blim Post Suggestion and Vote.

== Requirements ==
**General Features** may be used by anyone with any web server (Apache, NGINX, etc.).

== Plugin Features ==

= General Features =

* Suggests posts
1. As the heading implies, the plugin gets a post related by category to the current post and suggests to reader

* Upvote and Downvote
Readers can upvote or downvote on various posts.

== Installation =
= Through Dashboard =
1. Log in to your WordPress admin panel and go to Plugins -> Add New
2. Type Blim Post Suggestion and Vote in the search box and click on search button.
3. Find Blim Post Suggestion and Vote.
4. Then click on Install
5. Now, after that activate the plugin.


= Installing Via FTP =
1. Download the plugin to your hardisk.
2. Unzip.
3. Upload the blim-post-suggestion-and-vote folder into your plugins directory.
4. Log in to your WordPress admin panel and click the Plugins menu.
5. Then activate the plugin.



== Frequently Asked Questions ==

= Can I show only one feature? =

Yes, BPSV offers the ability to choose which feature is needed for your website. This can be set at the general admin settings under the "blim post suggestion and vote settings" sub menu. You can also decide to temporarily disable both features if you want, without uninstalling the plugin. 

= Is the BPSV for WordPress free? =

Yes, BPSV will always be free and open source.
